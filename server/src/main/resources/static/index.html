HTTP Interfaces

  • Debug interfaces for autocomplete and forward & reverse geocoding
  • REST-y GET interface at "/" like /?query=nyc or /?ll=40.74,-74.0.
  • Thrift-json GET/POST interfaces at /geocode, /reverseGeocode, /bulkReverseGeocode and /bulkSlugLookup that take pretty printed json requests as input
